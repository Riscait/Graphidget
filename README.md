# Graphidget
Graph & Widget for iOS.

## Set up
Install Mint.

Clone the project.

```
$ git clone https://github.com/Riscait/Graphidget.git
$ cd Graphidget
```

Run `make setup` .
After setup is complete, Project automatically opens in Xcode.
